This project demonstrates automation testing using Playwright (JavaScript) for the SwiftTranslator website.
The main objective is to validate the Singlish to Sinhala translation functionality by executing automated test cases and verifying the translated output.
The project was created as part of a QA / Software Testing assignment, following proper testing steps such as manual testing, test case documentation, automation execution, and version control using GitHub.

Tools & Technologies Used

Playwright – Automation testing framework
JavaScript – Test scripting language
Node.js – Runtime environment
Git & GitHub – Version control
SwiftTranslator Website – System under test
